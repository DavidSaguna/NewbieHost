  • Panel

  • How do I schedule automatic backups?

    Step 1. Access your server's panel

    Step 2. Click on the Schedules button

    Step 3. Click on "CREATE SCHEDULE"

    Now this should appear:

    Explaination: You can put any name you'd like. Number 1 is the setting of the frequency. An * means "Anytime", an */number means "Do it from number to number minutes/hours/months etc. A "Number" means "Do it only when the time is equal to this number." For example if I put 3 it will do it at 3AM. (GMT+2 time and 24hour format)

    Number 2 is simple to understand. If checked, the action will only execute when the server is online.

    Number 3 will turn on or off the schedule.

     

    After you created the schedule you will have to create an action that will execute when the schedule starts.

    Click on the schedule and then on "NEW TASK"

    Select the backup action from the dropdown menu. Now you can pick any files that you want to NOT include in the backup (such as plugins' jar files) and then press on "CREATE TASK"!

    Now it should look something like this: (Name, next run at and time interval can be different)

  • How do I get the IP for me and my friends?

    Step 1. Access your server's panel after you created it.

    Step 2. Look in the top left corner. There you should see a long number (The IP) that has 4 dots followed by a port. (Example: 123.45.678.91:25565)

    Step 3. Click on it to copy it and give it to your friends.

    Step 4. Introduce (Your friends too) the ip in Minecraft. Tip: If you copied it, press Ctrl and V at the same time to paste it in here:

  • After I got a minecraft server, how do I make it work?

    Note: Minecraft servers aren't available anymore. We will bring them back in the future!


    You now probably have your server and account but when you press on "START" it gives an error. That is because you haven't installed the server's jar. The jar is the server's software. Without it, the server can't run/start.

    So what would you want to do is to download a good jar that you'd like. You can get your jar file from getbukkit.org. It will show the latest version you can download.

    However, sometimes you would like to use other versions, for example 1.12.2 or 1.8 so what you should do is go to downloads (Top right corner) and select one of the three choices:

    Vanilla - Vanilla is the offical server jar released by Microsoft but you can't install plugins on it.

    Spigot - This is the most optimizated server jars provider. Their jars run perfectly and have good settings. You can install plugins too.

    Craftbukkit - This allows you to install plugins but some plugins don't have support for craftbukkit. Also, it is kind of discontinued, with Spigot having a lot more features and friendly developer API.

     

    After you decide which one you will use, click on one of the three buttons in the dropdown. Now you will have a long page of versions. They should look something like this:

    After you scroll to your version click the yellow download button to download it. It should be downloaded in your downloads folder.

    Now, go to your server and click on "FILE MANAGER" and upload it by dragging it from file explorer or from clicking the blue UPLOAD button. Here is a short video showing how to do it.

    The last thing you will want to do, is to rename the file to server.jar

    Now you can start your server! Have fun!
